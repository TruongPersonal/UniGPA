# 🎓 UniGPA

**UniGPA** là ứng dụng Flutter hiện đại giúp sinh viên Việt Nam theo dõi kết quả học tập, tính toán GPA và lập kế hoạch mục tiêu điểm số một cách khoa học và hiệu quả.

## ✨ Tính năng nổi bật

- **Quản lý học tập**: Tổ chức môn học theo Năm học và Học kỳ rõ ràng.
- **Tính GPA tự động**: Hỗ trợ hệ điểm 4.0 và 10.0, tự động quy đổi điểm chữ (A+, A, B+,...).
- **Theo dõi mục tiêu**: Nhập điểm quá trình để biết cần bao nhiêu điểm thi để đạt mục tiêu môn học.
- **Chế độ màn hình**: Hỗ trợ đầy đủ Giao diện Sáng (Light Mode) và Tối (Dark Mode) bền bỉ theo hệ thống.
- **Sao lưu & Phục hồi**: Xuất/Nhập dữ liệu qua file CSV để bảo mật thông tin cá nhân.
- **Quyền riêng tư**: Dữ liệu lưu trữ hoàn toàn cục bộ trên thiết bị (Local Storage - Hive), không cần internet.
- **Đa nền tảng**: Hoạt động mượt mà trên Android, iOS, macOS và Web.

## 🛠️ Công nghệ sử dụng

| Thành phần | Công nghệ |
|------------|-----------|
| **Framework** | Flutter (Dart) |
| **State Management** | Provider |
| **Local Database** | Hive (NoSQL) |
